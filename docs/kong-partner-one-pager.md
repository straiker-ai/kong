# Straiker + Kong: AI Security for Enterprise Gateways

- **Agentic workflows** — Multi-turn agent orchestration protection
- **Enterprise deployments** — Per-app policies, identity-aware audit
- **Multimodal flows** — Images, PDFs, external context scanning
- **Compliance** — Full audit trail linked to users, apps, tools
